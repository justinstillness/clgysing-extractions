**Detailed Description of Bootstrap Template (TPL-002)**\
\
The Bootstrap Template is designed to solve the \"memory problem\" by carrying essential context from one ephemeral AI session to the next [15]. It prescribes the following fields and sections:\
\
**1. Header & Identification**\
*   **Project Identification:** Full Name, Abbreviated Name, and Project Code [16].\
*   **Source Chat:** Name, Date, and Type (e.g., Execution, Review) of the session generating the bootstrap [16].\
*   **Target Chat:** Suggested name for the next session, Bootstrap Type (Continuation, Branch, or New Instance), and current Phase [16].\

**2. Session Requirements**\
